# mini-guia-educacao-financeira-ia
Desenvolvi um mini guia de Educação Financeira no NotebookLM utilizando fontes abertas confiáveis. Defini objetivos de estudo, criei perguntas estratégicas e testei diferentes prompts para gerar resumos, glossário e revisão. O projeto destaca o uso da IA como ferramenta de aprendizagem ativa e organização do conhecimento.
